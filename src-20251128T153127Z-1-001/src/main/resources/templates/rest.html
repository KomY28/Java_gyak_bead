NB1 Adatbázis API

RESTful API Dokumentáció

Az alkalmazás biztosít egy publikus API-t a klubok kezelésére.

Elérhető Végpontok:

Metódus URL Leírás
GET /api/klubok Összes klub listázása (JSON)
GET /api/klubok/{id} Egy adott klub lekérése ID alapján
POST /api/klubok Új klub létrehozása (Body: JSON)
DELETE /api/klubok/{id} Klub törlése

Tesztelés:

Kattints ide a lista lekéréséhez: /api/klubok megnyitása
